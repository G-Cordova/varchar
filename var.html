Var/Varchar Differences

What is the difference between CHAR and VARCHAR?

  1. CHAR is faster, VARCHAR is more flexible
  2. CHAR has the potential to waste bytes if all charactors are not utilized, where as with VARCHAR, no bytes are wasted because a (nul) is counted as the last stop byte
  3. CHAR ≤ 255 and VARCHAR ≤ 65535
  4. Expanded VARCHAR's do not index well
